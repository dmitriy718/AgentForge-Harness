# E3 Long Context Summary

Summarize this operating rule into a concise agent policy:

Models must not invent facts. They must mark unknowns, cite evidence, ask before destructive action, run validation, and preserve user changes. For code work, inspect the repository first and keep changes scoped. For system work, record current state before changing it and define rollback.

Return:

- policy,
- forbidden behaviors,
- validation checklist.
