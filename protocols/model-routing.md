# Model Routing Protocol

Use stronger models for ambiguity and higher risk. Use local models for speed, privacy, and repeatable low-risk work.

## Route To Codex

- Repository edits.
- Terminal work.
- Multi-file refactors.
- System diagnosis.
- Work requiring local tools.

## Route To A Local Model

- Drafting variants.
- Summarizing local docs.
- Extracting structured data from non-sensitive text.
- Running benchmark prompts.
- Privacy-sensitive first passes where lower accuracy is acceptable.

## Route To Multi-Agent Swarm

- Architecture choices.
- Security reviews.
- Complex debugging.
- Claims requiring independent challenge.
- Any task where a single confident answer is dangerous.
