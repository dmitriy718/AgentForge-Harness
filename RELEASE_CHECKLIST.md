# Release Checklist

- [ ] `aih doctor` passes.
- [ ] `python3 -m unittest` passes.
- [ ] `aih release "candidate name"` creates a release packet.
- [ ] `aih list` shows all templates and protocols.
- [ ] `HOW_TO_USE.md` is current.
- [ ] Codex skill validates.
- [ ] At least one run directory demonstrates the workflow.
- [ ] Production readiness protocol is satisfied.
- [ ] Release packet includes doctor output, manifest, self-test log, validation notes, known limitations, support note, and example prompt.
- [ ] Known limitations are documented.
- [ ] No secrets or personal credentials are included.
- [ ] Scripts are executable.
- [ ] Final package can be explained in under 60 seconds.
