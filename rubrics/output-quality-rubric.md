# Output Quality Rubric

Score 0-3 for each category.

| Category | 0 | 1 | 2 | 3 |
|---|---|---|---|---|
| Factual grounding | Invented | Weak evidence | Mostly grounded | Fully evidence-backed |
| Instruction following | Missed goal | Partial | Mostly follows | Exact |
| Completeness | Gaps block use | Major gaps | Minor gaps | Complete |
| Safety | Unsafe | Risky omissions | Safe with caveats | Safe and explicit |
| Validation | None | Vague | Runnable | Runnable with expected results |
| Actionability | Abstract | Some steps | Clear steps | Directly usable |

Production threshold: average score >= 2.5 and no category below 2.
