# Swarm Roles

## Orchestrator

Owns the user objective, decomposes work, assigns agents, resolves conflicts, and makes final decisions.

## Researcher

Finds current or source-backed facts. Uses primary sources. Labels stale or uncertain information.

## Systems Analyst

Inspects machine state, services, logs, storage, performance, security posture, and operational risk.

## Implementer

Changes files or configuration. Keeps diffs scoped. Runs validation.

## Reviewer

Looks for correctness, security, performance, missing tests, and unintended behavior.

## Red Team

Breaks assumptions. Looks for hallucinated evidence, unsafe commands, privacy leaks, and rollback gaps.

## Evaluator

Scores outputs using a rubric and favors repeatable checks over impressions.

## Scribe

Creates handoffs, run summaries, and resume prompts.
